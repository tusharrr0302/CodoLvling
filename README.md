# Codo Lvling

> A gamified DSA learning platform inspired by RPG progression systems, multiplayer battles, and interactive coding adventures.

![Codo Lvling Banner](./public/CodoLvling.webp)

---

# Overview

Codo Lvling transforms traditional coding practice into an immersive game-like experience where users level up, battle enemies, unlock regions, earn rewards, and compete with other players while improving their Data Structures & Algorithms skills.

The platform combines:
- Gamification
- Multiplayer coding battles
- AI-powered mentorship
- Interactive DSA visualizers
- Community-driven learning

into a single full-stack learning ecosystem.

---

# Features

## RPG-Style Progression System
- XP and level progression
- Rank tiers and achievements
- Coins and rewards
- Daily streak tracking
- Unlockable regions and challenges

---

## Multiplayer Coding Battles
- Real-time 1v1 battles
- Duo co-op mode
- 2v2 team battles
- Live matchmaking with Socket.io
- Performance-based scoring system

---

## AI-Powered Code Evaluation
Integrated Gemini AI for:
- Test case validation
- Code quality analysis
- Complexity evaluation
- Personalized mentorship feedback
- Optimization hints and suggestions

---

## Interactive DSA Visualizers
Visual learning support for:
- Arrays
- Trees
- Graphs
- Linked Lists
- Dynamic Programming
- Recursion
- Stacks & Queues
- Strings

## Community System
- Discussion forums

## Gamified World Exploration
- Story-based progression
- Region unlocking

# Architecture

## Frontend
The frontend is built using React.js with a modular component-based architecture.

Key systems include:
- Multiplayer Arena
- Logic Lab
- Community Hub
- World Map
- Battle Engine
- Story Engine
- DSA Visualizers

---

## Backend
The backend handles:
- Authentication
- Matchmaking
- Real-time battles
- AI evaluation
- Community APIs
- Progress synchronization
- Inventory systems

Socket.io powers the real-time multiplayer infrastructure.
